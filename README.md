# Atividade avaliativa 07 - IFRN

## Instruções

- Instale o python3;
- Instale localmente o ollama;
- Traga a imagem de um modelo de LLM para a sua máquina:
```bash
ollama pull llama2
```
- Coloque um arquivo de audio na pasta raíz
- Rode o projeto localmente
